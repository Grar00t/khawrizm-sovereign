## 🟢 القسم العربي — المصدر الأصلي

### ما هو HAVEN؟

**HAVEN** هو بيئة تطوير مدمجة بالذكاء الاصطناعي، مبنيّة من الصفر في المملكة العربية السعودية، تحت مفهوم **السيادة الرقمية الكاملة**. لا يتسرّب أي بيانات خارج حدود نظامك. الذكاء الاصطناعي يعمل محليًا. الكود يبقى ملكك.

يُجسّد HAVEN إرث **محمد بن موسى الخوارزمي** — العالِم المسلم الذي أرسى أُسس الخوارزميات والجبر — في بيئة تطوير عصرية تُقدّم اللغة العربية كلغة أُولى، وتضمن السيادة الرقمية كواجب لا كخيار.

> **بنى سليمان الشمري (أبو خوارزم / @Grar00t) هذا المشروع من الصفر — ليُثبت أن المطوّر العربي قادر على صناعة العالم، لا مجرد استهلاكه.**

---

### ✨ المميزات الرئيسية

#### 🧠 الذكاء الاصطناعي ثلاثي الفصوص (Three-Lobe AI)
- **الفص المعرفي (Cognitive Lobe):** الفهم العميق للسياق، التحليل الدلالي، توليد الكود
- **الفص التنفيذي (Executive Lobe):** التخطيط متعدد الخطوات، التنسيق، تتبع الأهداف
- **الفص الحسّي (Sensory Lobe):** معالجة المدخلات، تحليل الكود الفوري، اكتشاف الأنماط

#### 🌐 محرك نيّة (NiyahEngine) — معالجة اللغة العربية أولًا
- **2900+ جذر عربي** في قاموس الجذور
- **7 لهجات** عربية: الفصحى، النجدية، الحجازية، المصرية، الشامية، العراقية، المغربية
- **8 نبرات** نصية: رسمي، غير رسمي، تقني، تحليلي، إبداعي، تعليمي، تجاري، أكاديمي
- **9 مجالات** متخصصة: تطوير البرمجيات، الأمن السيبراني، الذكاء الاصطناعي، البيانات، الشبكات، قواعد البيانات، DevOps، الواجهات، الخوارزميات

#### 🔐 الأمان والسيادة
- تشفير **AES-256-GCM** لجميع الجلسات
- توقيع رقمي **Ed25519**
- تجزئة كلمات المرور **Argon2id**
- **TLS 1.3** لجميع الاتصالات
- **SovereignBridge:** عزل العمليات وإدارة أجهزة QEMU الافتراضية
- **SovereignSessionCleaner:** تنظيف آمن ومشفّر للجلسات

#### 🚀 مُوجِّه النماذج (ModelRouter)
- دعم **16 عائلة نموذج** من Ollama
- تسجيل ديناميكي للنماذج
- ذاكرة تخزين LRU (512 إدخال)
- نظام تسجيل نقاط لاختيار أفضل نموذج

#### ✍️ مُكمِّل النيّة (NiyahCompletionProvider)
- **5 طبقات** من الاكتمال الذكي داخل المحرر
- اقتراحات مدركة للسياق
- دعم كامل لـ Monaco Editor

## 🟢 English Section

### What is HAVEN?

**HAVEN** is an AI-integrated development environment built from the ground up in **Saudi Arabia** under the principle of **complete digital sovereignty**. No data leaks outside your system boundary. AI inference runs locally. Your code belongs to you — and only you.

HAVEN embodies the legacy of **Muhammad ibn Musa Al-Khwarizmi** — the 9th-century Muslim scholar who laid the foundations of algorithms and algebra — reimagined as a modern IDE that treats Arabic as a first-class language and digital sovereignty as a duty, not a feature.

> **Sulaiman Alshammari (أبو خوارزم / @Grar00t) built this project from zero to prove that the Arab developer is a creator of the world, not merely a consumer of it.**

---

### ✨ Core Features

#### 🧠 Three-Lobe AI Architecture
A biologically-inspired cognitive framework that parallels human brain organization:

| Lobe | Role | Capability |
|------|------|------------|
| **Cognitive** | Deep understanding | Semantic analysis, code generation, context modeling |
| **Executive** | Planning & control | Multi-step planning, goal tracking, coordination |
| **Sensory** | Input processing | Real-time code analysis, pattern detection, stream handling |

All three lobes execute in **parallel** via Ollama, with the ThreeLobeAgent orchestrating synchronization and result fusion.

#### 🌐 NiyahEngine — Arabic-First NLP
The most sophisticated Arabic NLP engine ever embedded in an IDE:

- **2,900+ Arabic root entries** covering Classical and Modern Standard Arabic
- **7 dialect detection models:**
  - Fus'ha (Modern Standard Arabic / Classical)
  - Najdi (Central Arabian Peninsula)
  - Hijazi (Western Saudi / Hejaz)
  - Misri (Egyptian)
  - Shami (Levantine)
  - Iraqi (Mesopotamian)
  - Maghrebi (North African)
- **8 tone classifiers:** Formal, Informal, Technical, Analytical, Creative, Instructional, Commercial, Academic
- **9 domain routers:** Software Development, Cybersecurity, AI/ML, Data Science, Networking, Databases, DevOps, UI/UX, Algorithms
- **Intent graph** with weighted edges and cluster analysis (200-session lookback)

#### 🔐 Sovereignty & Security Stack

```
┌─────────────────────────────────────────────────────┐
│               HAVEN Security Architecture           │
├─────────────────────────────────────────────────────┤
│  Data at Rest    │  AES-256-GCM  │  Per-session keys│
│  Data in Transit │  TLS 1.3      │  Certificate pin │
│  Auth            │  Ed25519      │  Zero-trust model│
│  Key Derivation  │  Argon2id     │  Memory-hard KDF │
│  Process Isolation│ SovereignBridge│ QEMU sandboxing│
│  Session Cleanup │ SessionCleaner│ Cryptographic wipe│
├─────────────────────────────────────────────────────┤
│  Compliance: PDPL · SDAIA · NCA-ECC · NIST SP 800  │
└─────────────────────────────────────────────────────┘
```

#### 🚀 ModelRouter — Intelligent Model Dispatch

- **16 model families** supported (Llama 3.x, Mistral, DeepSeek, Phi-3, Gemma 2, Qwen, CodeLlama, Falcon, Orca, WizardCoder, StarCoder, Mixtral, Command-R, Yi, Solar, Nous-Hermes)
- Dynamic scoring based on task type, token budget, latency targets
- **LRU Cache:** 512 entries (upgraded from 64), TTL-based expiration, memory-aware eviction
- Automatic model fallback chain

#### ✍️ NiyahCompletionProvider — 5-Layer Inline Completion

| Layer | Description |
|-------|-------------|
| 1. **Syntactic** | Language grammar rules, bracket completion |
| 2. **Semantic** | Type inference, symbol resolution |
| 3. **Contextual** | File-level and project-level patterns |
| 4. **Intent** | NiyahEngine intent-driven prediction |
| 5. **Lobe-Fused** | Cross-lobe consensus completions |

### 🔒 Security Policy

HAVEN follows a **Zero-Trust Sovereignty** model:

1. **No outbound AI calls** — All inference is local via Ollama
2. **Encrypted sessions** — Every session encrypted with AES-256-GCM using a unique key
3. **Process isolation** — Terminal execution sandboxed via SovereignBridge
4. **Cryptographic session cleanup** — SessionCleaner overwrites memory before deallocation
5. **Audit logging** — All AI interactions logged locally with tamper-evident signatures

To report a security vulnerability, email **security@khawrizm.com** with subject `[HAVEN-SEC]`.

# Contributing to KHAWRIZM / المساهمة في خوارزم

> السيادة الرقمية مسؤولية جماعية — Digital sovereignty is a collective responsibility

## 🇸🇦 العربية

### كيف تساهم

1. **Fork** المستودع
2. أنشئ فرع جديد: `git checkout -b feature/اسم-الميزة`
3. اكتب الكود واختبره محلياً
4. تأكد من:
   - عدم وجود أي telemetry أو اتصال سحابي
   - التوافق مع PDPL
   - دعم اللغة العربية إذا كانت الميزة تواجه المستخدم
5. افتح Pull Request مع وصف واضح

### قواعد الكود

- **TypeScript strict mode** دائماً
- **Zero cloud dependencies** — كل شيء محلي
- **التعليقات** بالعربية والإنجليزية للملفات الأساسية
- **الاختبارات** مطلوبة لكل ميزة جديدة (vitest)
- **لا telemetry** — أي كود يرسل بيانات خارجية سيُرفض فوراً

### مجالات المساهمة المطلوبة

- 🔤 توسيع قاعدة الجذور العربية (ArabicRootsExpanded.ts)
- 🌍 ترجمات جديدة (i18n.ts)
- 🧪 اختبارات إضافية
- 🛡️ تحسينات أمنية
- 📖 توثيق بالعربية

## 🇬🇧 English

### How to Contribute

1. **Fork** the repository
2. Create a branch: `git checkout -b feature/feature-name`
3. Write code and test locally
4. Ensure:
   - Zero telemetry or cloud connections
   - PDPL compliance
   - Arabic language support for user-facing features
5. Open a Pull Request with clear description

### Code Standards

- **TypeScript strict mode** always
- **Zero cloud dependencies** — everything local
- **Comments** in Arabic + English for core files
- **Tests** required for all new features (vitest)
- **No telemetry** — any code sending external data will be rejected immediately

### Areas Needing Contribution

- 🔤 Expanding Arabic root database (ArabicRootsExpanded.ts)
- 🌍 New translations (i18n.ts)
- 🧪 Additional tests
- 🛡️ Security improvements
- 📖 Arabic documentation
